# WindSurf 管理后台 UI 指南

## 概述

WindSurf 管理后台是一个基于 React + Ant Design 构建的现代化 Web 管理系统，用于管理 WindSurf 续杯工具的激活码、账号和用户。

## 技术栈

- **前端框架**: React 18
- **UI 组件库**: Ant Design 5.x
- **路由**: React Router 6
- **图表**: Recharts
- **HTTP 客户端**: Axios
- **日期处理**: Day.js
- **构建工具**: Create React App + CRACO

**账号状态**:
- 🟢 `green` - 可用
- ⚫ `gray` - 停用
- 🔵 `blue` - 使用中

**分配状态**:
- 🟠 `orange` - 已分配
- 🟢 `green` - 未分配

## 页面组件

### 1. 登录页 (`Login.js`)

居中卡片式登录表单，包含：
- 系统 Logo 和标题
- 用户名输入框
- 密码输入框
- 登录按钮
- 默认账号提示

### 2. 仪表盘 (`Dashboard.js`)

**统计卡片区 (4列)**:
```
┌──────────┐ ┌──────────┐ ┌──────────┐ ┌──────────┐
│ 总激活码  │ │ 总账号数  │ │ 活跃用户  │ │ 已提号次数│
│   156    │ │   89     │ │   42     │ │   328    │
│ 活跃/过期 │ │ 可用/分配 │ │ 当前在线  │ │ 累计总数  │
└──────────┘ └──────────┘ └──────────┘ └──────────┘
```

**图表区 (2列)**:
- 左侧：使用趋势折线图 (提号/切号)
- 右侧：最近操作日志表格

### 3. 激活码管理 (`ActivationCodes.js`)

**操作栏**:
- 搜索框
- 状态筛选下拉
- 生成激活码按钮

**表格列**:
| 激活码 | 状态 | 提号限制 | 有效期 | 活跃用户 | 创建时间 | 操作 |

**弹窗**:
- 生成激活码表单
- 激活码详情查看

### 4. 账号管理 (`Accounts.js`)

**统计卡片区 (4列)**:
```
┌──────────┐ ┌──────────┐ ┌──────────┐ ┌──────────┐
│ 总账号数  │ │ 可用账号  │ │  已分配   │ │  使用率   │
│   89     │ │   45     │ │   44     │ │  49.4%   │
└──────────┘ └──────────┘ └──────────┘ └──────────┘
```

**操作栏**:
- 搜索框
- 状态筛选
- 分配状态筛选
- 批量导入按钮
- Token添加按钮
- 批量重置分配按钮
- 添加账号按钮

**表格列**:
| 邮箱 | 名称 | API Key | 状态 | 分配状态 | 分配给 | 创建时间 | 操作 |

### 5. Token导入 (`TokenImport.js`)

- 功能说明 Alert
- Token 输入文本域
- 保存/清空按钮
- 注意事项 Alert
- Token 格式示例

### 6. 管理员管理 (`AdminManagement.js`)

**表格列**:
| 用户名 | 邮箱 | 角色 | 状态 | 最后登录 | 创建时间 | 操作 |

**角色标签**:
- 🔴 `red` - 超级管理员
- 🔵 `blue` - 管理员

### 7. 系统配置 (`SystemConfig.js`)

**配置表单**:
- 激活码前缀 (Input)
- 激活码长度 (InputNumber: 8-32)
- 默认有效天数 (InputNumber: 1-365)
- 默认最大账号数 (InputNumber: 1-100)

**权限提示**: 只有超级管理员可修改

## 通用组件

```jsx
<div className="page-header">
  <h2>页面标题</h2>

```

### 2. 表格操作栏
```jsx

  <Space>

  <Space>
    {/* 右侧：操作按钮 */}
  </Space>
```

### 3. 统计卡片
```jsx
<Card className="dashboard-stat-card">
  <Statistic
    title="标题"
    value={数值}
    prefix={<图标 />}
    valueStyle={{ color: '#颜色' }}
  />
  <div className="stat-desc">描述文字</div>
</Card>
```

## 响应式设计

- 侧边栏可折叠 (`collapsed` 状态)
- 表格支持水平滚动 (`scroll={{ x: width }}`)
- 表单最大宽度 600px
- 卡片使用 `gutter={16}` 间距
